Cascading Style Sheets

Folha de Estilo em Cascata

CSS é uma linguagem de folha de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML (incluindo dialetos XML como SVG, MathML ou XHTML). O CSS descreve como os elementos devem ser renderizados na tela, no papel, na fala ou em outra mídia.
